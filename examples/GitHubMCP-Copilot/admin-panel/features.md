# Admin Panel Features

## Dashboard

The main dashboard shows:
- All agents and their current status (active / inactive)
- Guardrail alert count for the last 24 hours
- Current risk level from the most recent scanner run
- Last activity timestamp for each active agent

## Agent Management

From the Agents tab:
- Toggle any agent on or off
- View the agent's definition, config, and test checklist
- See recent activity logs for each agent
- Trigger a test run to verify an agent is responding correctly

## Guardrail Alerts

From the Alerts tab:
- View all active alerts in chronological order
- Filter by severity, agent, or rule category
- Mark alerts as reviewed
- See the full context of what triggered each alert

## Risk Profile

From the Risk tab:
- View the current risk profile generated by the scanner
- See when the last scan was run
- Trigger a new scan (points to the same target as the initial setup)
- Review individual findings and their status (active / dismissed)

## File Upload

From the Settings tab:
- Upload a new agent definition markdown file to update an existing agent
- Upload a new template to add it to the templates library
- Upload a custom risk rule to add it to the scanner's rule set

See [upload-spec.md](upload-spec.md) for file format requirements.

## Configuration View

From the Settings tab:
- View the current `config/platform.json` in a readable format
- See which agents are active and their key configuration values
- Link out to edit config files directly (no in-panel editing - by design)
