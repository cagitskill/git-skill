Steps to Connect
Connection Pipeline — IDE to GitHub
📄
Configure mcp.json
Add the GitHub MCP server URL to your IDE's config. Root key is "servers" — not "mcpServers".
VS Code · Visual Studio · JetBrains
🔑
Authenticate
OAuth one-click for GitHub.com. PAT with scoped permissions for GitHub Enterprise Cloud with data residency.
OAuth recommended · PAT for GHE
🤖
Switch to Agent Mode
MCP tools only appear in Agent mode. Select it from the Copilot Chat mode dropdown — tools are invisible in Ask or Edit mode.
Agent mode · Tools icon visible
💬
Prompt in Natural Language
Ask Copilot to create an issue, list open PRs, search code, or fetch commit history. No commands — just plain English.
Natural language → GitHub action
⚡
GitHub MCP Tool Called
Copilot selects the right tool automatically. A confirmation dialog appears before any write operation executes.
Auto tool selection · Confirm writes
✅
GitHub Data in Your IDE
Results surface directly in Copilot Chat. No browser tab switching. Your full GitHub context, right where you code.
Zero context switching

What the GitHub MCP Server Gives Copilot Access To
📁
Repository Content
Read files, search code, fetch commit history from any repo you have access to
🐛
Issues
Create, update, label, assign, and comment on issues without leaving your editor
🔁
Pull Requests
Open PRs, list reviews, add comments, and check merge status from Copilot Chat
🔍
Code Search
Search across all files in a repo or the entire GitHub index instantly
📊
Commit History
Fetch recent commits, diff specific changes, trace who changed what and when
🌿
Branch Management
List branches, check status, understand what's on each branch — all from chat
👤
User & Org Context
Access assigned issues, your open PRs, org-level repo lists with the right scopes
🔔
Notifications
Surface what needs your attention — reviews requested, mentions, CI failures
