Copilot Quiz — Editing Guide

A reference for anyone updating questions, answers, or prize eligibility in copilot-quiz.html

Where to find everything

All quiz content lives in a single JavaScript object near the top of the file. Open copilot-quiz.html in any text editor and search for warmup: [ — that's the start of the question data. The four sections appear in this order:

Section Search for Prize eligible?
Warm Upwarmup: [Never
Beginnerbeginner: [Every other question
Intermediateintermediate: [Every other question
Advancedadvanced: [Every other question

What a question looks like

Open-ended question (no choices)
{
  q: "What is GitHub Copilot?",
  mc: false,
  swag: true,
  answer: "GitHub Copilot is an AI-powered coding assistant..."
},
Multiple choice question
{
  q: "Which of the following best describes how Copilot generates suggestions?",
  mc: true,
  swag: false,
  choices: [
    "A. It searches Stack Overflow in real time",
    "B. It uses an AI model trained on publicly available code",
    "C. It copies code from your current repo",
    "D. It runs a local model on your machine"
  ],
  answer: "B — It uses an AI model trained on publicly available code."
},

Editing questions

Find the question you want to change, then update the text inside the quotes after q:. Keep the surrounding quotes and trailing comma intact.

Don't change the structure — just swap the text. If the original ends with a ?, your replacement should too.

Editing answers

The answer text appears after answer:. You can make it as long or short as needed. Answers are revealed when the host clicks the card during the session — participants don't see them automatically.

Editing multiple choice options

Choices are listed inside square brackets [ ], each option in quotes, separated by commas. Keep the A. B. C. D. labels — they display as-is in the quiz.

If you remove a choice, make sure to remove the trailing comma on the line before it. If you add a choice, add a comma after the previous last item.

Editing prize eligibility

Each question has a swag: field set to either true or false.

Value What it does
swag: true 🎁 Prize eligible badge appears on the card
swag: false No badge — question is not prize eligible

Current rules baked into the file:

  • Warm Up: all set to false — these are icebreakers, not prize questions
  • Beginner, Intermediate, Advanced: every other question is true, starting with the first
To make a specific question prize eligible, change its swag: false to swag: true. To remove eligibility, do the reverse. Warm Up questions should stay false.

Adding a new question

Copy an existing question block (everything from { to },), paste it where you want it in the list, and update the fields. Make sure every question in the list except the last one ends with a comma after the closing }.

Removing a question

Delete the entire block from the opening { to the closing }, including the comma. Then check that the question now above where it was doesn't have a trailing comma if it's the last item in the section.
